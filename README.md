## Getting started

### Prerequisites

- Ubuntu 20.04
- Ros Noetic
- ROS1
- TurtleBot3 packages
- TurtleBot3 Burger
- Python3
- git

### Clone the project

```bash
$ cd ~/catkin_ws/src
$ git clone git@github.com:AuE893-Spring21-Team-2/aue_finals.git
```

### Build code in your catkin workspace

```bash
$ cd ~/catkin_ws
$ catkin_make
$ source ~/catkin_ws/devel/setup.bash
```

### Make the node executable

```bash
$ cd ~/catkin_ws/src/aue_finals/src/scripts
$ chmod u+x *.py
```
